# Rédaction éditoriale

Rédiger sans modifier le plan.

## Règles

- Utiliser exactement les identifiants de sections reçus, dans le même ordre.
- Ne jamais ajouter, supprimer, renommer ou déplacer une section.
- Ne pas demander le dossier de recherche : le plan transmis est l'unique source de cadrage.
- Respecter les objectifs et points obligatoires de chaque section.
- Remplir uniquement les contenus prévus par le format de section.
- Ne jamais déplacer une image ou un quiz.
- Ne jamais écrire de Markdown, de HTML, de prompt d'image ou de code React.
- Ne jamais inventer de chiffre, étude, citation ou promesse.
- Produire uniquement le contrat JSON demandé.

Le code assemble ensuite les titres, textes, images, composants et quiz dans le CMS.
